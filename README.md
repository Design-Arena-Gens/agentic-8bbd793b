# استوديو المونتاج (Agentic Video Studio)

أداة مونتاج تعمل داخل المتصفح لبناء فيديوهات MP4 عن طريق قص وترتيب عدة مقاطع محلية، معتمدة على FFmpeg WebAssembly.

## المزايا الرئيسية

- استيراد عدة مقاطع فيديو (MP4, MOV, WEBM, ...).
- قص البداية والنهاية لكل مقطع مع إمكانية الالتقاط من موضع التشغيل الحالي.
- إعادة ترتيب المقاطع بسهولة للوصول إلى التسلسل الزمني المطلوب.
- دمج المقاطع في فيديو واحد بصيغة MP4 (H.264 + AAC) مع تجهيز `faststart` لتسريع التحميل.
- يعمل بالكامل داخل المتصفح دون رفع الملفات أو الحاجة لبرامج خارجية.

## تشغيل محلي

```bash
npm install
npm run dev
```

انتقل إلى [http://localhost:3000](http://localhost:3000) للاستعمال.

## بناء للإنتاج

```bash
npm run build
npm run start
```

## ملاحظات تقنية

- يستخدم المشروع Next.js 16 مع واجهة App Router.
- تتم معالجة الفيديو عن طريق مكتبة `@ffmpeg/ffmpeg` (WebAssembly) وتخزين الوسائط مؤقتاً داخل نظام الملفات الظاهري للمتصفح.
- تم ضبط التصميم ليدعم الاتجاه من اليمين إلى اليسار ويتضمن خطوط عربية (`Noto Kufi Arabic`).

## رخصة الاستخدام

هذا المشروع مخصص لأغراض تجريبية ويمكن التوسع فيه بحرية حسب الحاجة.
